# 🔒 Privacy Policy - HTML แบบง่าย

## 📋 สรุปการติดตั้ง

ได้สร้างหน้า Privacy Policy แบบ HTML ธรรมดาๆ ที่ไม่ต้องเชื่อมต่อกับระบบอะไรมาก แค่เป็นหน้าแสดงผลเฉยๆ

## 🌐 URL ที่สามารถเข้าถึงได้

```
https://yourdomain.com/privacy-policy.html
```

หรือสำหรับการทดสอบในเครื่อง:
```
http://localhost:5000/privacy-policy.html
```

## 📁 ไฟล์ที่สร้าง

### ไฟล์ใหม่
- `backend/public/privacy-policy.html` - หน้า Privacy Policy แบบ HTML ธรรมดา

## ✨ คุณสมบัติ

### 🎨 การออกแบบ
- **Responsive Design** - รองรับทุกขนาดหน้าจอ
- **Modern UI** - ใช้ gradient และ shadow สวยงาม
- **Thai Language** - ภาษาไทยทั้งหมด
- **Easy Navigation** - มีปุ่มกลับหน้าแรก

### 📱 การใช้งาน
- **HTML ธรรมดา** - ไม่ต้องเชื่อมต่อกับระบบอะไรมาก
- **Static File** - แค่ไฟล์ HTML แสดงผลเฉยๆ
- **Fast Loading** - โหลดเร็วเพราะเป็นไฟล์ธรรมดา
- **No Dependencies** - ไม่ต้องพึ่งพา JavaScript หรือ CSS ภายนอก

### 📝 เนื้อหาที่ครอบคลุม

1. **ข้อมูลที่รวบรวม** - ส่วนบุคคล, เทคนิค, การใช้งาน
2. **วิธีการรวบรวมข้อมูล** - โดยตรงและอัตโนมัติ
3. **วัตถุประสงค์ในการใช้ข้อมูล** - บริการ, ปรับปรุง, การตลาด
4. **การแบ่งปันข้อมูล** - Rabbit Gateway, Google Cloud, MongoDB
5. **การรักษาความปลอดภัยข้อมูล** - เข้ารหัส, ควบคุมการเข้าถึง
6. **สิทธิ์ของคุณ** - เข้าถึง, แก้ไข, ลบข้อมูล
7. **การใช้ Cookies** - ประเภทและการจัดการ
8. **การเก็บรักษาข้อมูล** - ระยะเวลาและการลบ
9. **การโอนย้ายข้อมูล** - ระหว่างประเทศ
10. **การเปลี่ยนแปลงนโยบาย** - การแจ้งเตือน
11. **การคุ้มครองผู้เยาว์** - ข้อจำกัดอายุ
12. **ข้อมูลติดต่อ** - อีเมล, เบอร์โทร, ที่อยู่

## 🔧 การใช้งาน
